# Plantilla: Estructura por Miembro del Equipo

Dentro de cada semana, crea una carpeta con tu nombre:

```
semana-01/
├── tu-nombre/
│   ├── ejercicios/
│   │   ├── ejercicio-1.py
│   │   ├── ejercicio-2.py
│   │   └── ...
│   ├── notas.md
│   └── README.md (opcional)
├── otro-miembro/
│   ├── ejercicios/
│   └── notas.md
└── README.md (objetivos y temas de la semana)
```

## Ejemplo: Tu carpeta personal

```
tu-nombre/
├── ejercicios/
│   ├── ej1_variables.py
│   ├── ej2_condicionales.py
│   └── ej3_bucles.py
├── notas.md
└── README.md
```

**notas.md** debe contener:

- Conceptos aprendidos
- Dudas o dificultades
- Ejemplos importantes
- Links útiles
